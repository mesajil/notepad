# Set up

- Save the script setup_prettier.sh in the root of your project.
- Alternatively you should add execute permissions: chmod +x setup_prettier.sh
- Execute the file:

  ```
  sh setup_prettier.sh
  ```
  or
  ```
  ./setup_prettier.sh
  ```
